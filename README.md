# BookStore
Web API Project using .NET Core 3.1

## Technologies
- .NET Core 3.1
- Entity Framework 3.1
- Fluent API
- AutoMapper
- Swagger
- SQL Server

### Unit tests
- xUnit
- Moq
- SQLite In-Memory database

## Architecture
- 3 Layers:
  - Application layer (API)
    - Controllers
    - Dtos
  - Domain layer
    - Models
    - Interfaces
    - Services
  - Infrastructure layer
    - Repository Pattern
